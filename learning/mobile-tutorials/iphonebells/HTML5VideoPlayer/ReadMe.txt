### HTML5VideoPlayer ###

===========================================================================
DESCRIPTION:

This sample demonstrates how to leverage features from CSS3 and HTML5 to build a video controller using the <video> element, the HTMLMediaElement DOM interface and CSS3 Transitions and Transforms. The video controller allows pausing and playing, muting, volume control, monitoring of loaded data and seeking throughout the content in real-time.

===========================================================================
BUILD REQUIREMENTS:

Safari 4.0

===========================================================================
RUNTIME REQUIREMENTS:

Safari 4.0

===========================================================================
PACKAGING LIST:
